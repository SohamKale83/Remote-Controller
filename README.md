# Remote-Controller
A programmable handheld remote compatible with both gaming consoles and unmanned aerial vehicles.

## Hardware Required

| Component | Quantity | Notes |
|---|---|---|
| Arduino Nano | 2 | 1 for transmitter, 1 for receiver |
| NRF24L01+PA+LNA module | 2 | Long-range variant with external antenna |
| Analog Joystick module | 2 | X/Y axis + push button |
| Battery (specify type/voltage) | 2 | Power source for TX and RX |
| Jumper wires | — | Male-to-male / male-to-female as needed |
| Breadboard / PCB | 2 | For prototyping or final build |
| Antenna | 2 | Usually included with PA+LNA module |
